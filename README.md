 class類別，就像一個模組，可以產出具有相似特性的實體(物件)
 例如class動物
 第一隻動物是名叫"Nico"(名字)的"狗"(物種)
 第二隻動物是名叫"Nana"(名字)的"貓"(物種)
 他們都會有相同的"屬性"、不同的"參數"
 譬如，他們都會有"名字"、"物種"的屬性
 每隻動物對應到這個屬性都會有一個特殊的值("參數")
 Ex.
 class Animal():
  def __init__(self, name):
   self.name = name
 a = Animal("dog")  #建立一個名叫dog的Animal實體(物件)
 print(a.name)
